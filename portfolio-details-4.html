개인프로젝트 - 로또번호추천 사이트

  • 사용기술: JAVA, springboot, JSP, JSTL
  • 개발환경: Spring-tool-suite, Maven
  • 프로젝트 기간: 2022.06
  • URL: https://github.com/AgLine/lotto
    http://agline.cafe24.com/lotto

개발내용

사용자가 선택한 개수만큼 1~5개의 로또 번호를 추천해 주는 서비스를 개발했습니다. 랜덤 번호 생성과 중복 방지를 통해 신뢰성을 확보하고, 직관적인 결과 출력과 예외 처리를 통해 사용자 편의성을 향상했습니다. 최종적으로 서비스를 카페24 호스팅 환경에 배포했습니다.

  • 랜덤번호생성 :
    1~45 사이의 번호를 중복 없이 랜덤으로 생성
    사용자 지정 범위(1~5개)에 따라 결과 제공
  • 결과값 출력 :
    추천된 번호를 Table 형태로 시각화하여 깔끔하게 출력
  • 예외처리 :
    로또 번호의 개수를 선택하지 않고 번호 생성을 시도할 경우, 경고창을 통해 안내 메시지 제공
  • 배포 :
    완성된 서비스를 카페24 호스팅 환경에 배포하여 웹 상에서 접근 가능하도록 설정
