개인프로젝트 - 로또번호추천 사이트

  • 사용기술: JAVA, springboot, JSP, JSTL
  • 개발환경: IntelliJ, Maven, Amazon Linux EC2
  • 프로젝트 기간: 2022.06
  • URL: https://github.com/AgLine/lotto
    http://lotto.lineee.shop/

개발내용

사용자가 원하는 수량(1~5개)의 로또 번호 세트를 랜덤으로 추천해 주는 웹 서비스를 개발했습니다. 번호 중복 방지를 통해 신뢰성을 높였고, 직관적인 테이블 형태의 결과 출력과 예외 처리로 사용자 편의성을 고려했습니다. 완성된 서비스는 AWS EC2 환경에 배포하고, 가비아에서 구입한 도메인을 AWS Route 53을 통해 연결하여 퍼블릭하게 접근 가능하도록 구성했습니다.

  • 랜덤번호생성 :
    1~45 사이의 번호를 중복 없이 랜덤으로 생성
    사용자 지정 범위(1~5개)에 따라 결과 제공
  • 결과값 출력 :
    추천된 번호를 Table 형태로 시각화하여 깔끔하게 출력
  • 예외처리 :
    로또 번호의 개수를 선택하지 않고 번호 생성을 시도할 경우, 경고창을 통해 안내 메시지 제공
  • 배포 :
    AWS EC2 인스턴스에 WAR 패키지 형태로 배포. Tomcat 10을 사용해 실행
  • 도메인 :
    가비아에서 구입한 도메인을 AWS Route 53에 연결하여 자체 주소(http://lotto.lineee.shop)로 접속 가능하도록 설정
