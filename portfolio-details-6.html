팀프로젝트 - 전기차충전소ERP

  • 사용기술: Springboot2.7, nexacro N
  • 개발환경: Eclipse, Maven, Oracle Database 11g, TOBESOFT Nexacro Studio
  • 프로젝트 기간: 2024.08 ~ 2024.10
  • URL: https://github.com/AgLine/Nexacro

개발내용

  • 권한관리 :
    사용자 권한을 시스템관리자, 관리자, 일반 사용자, AS 기사로 분류하고, 각 권한에 따라 접근 가능한 메뉴를 동적으로 제어함
    Spring Boot에서 로그인 시 사용자 권한을 Oracle DB에서 조회하여nexacro.setEnvironmentVariable()을 통해 세션에 저장하고, Nexacro 화면 로딩 시 이 권한 정보를 기반으로 접근 가능한 메뉴를 동적으로 활성화 또는 비활성화
  • 콤보박스 코드 관리 :
    콤보박스 항목을 공통 코드 테이블로 분리하여 관리하며, 관리자만 접근 가능한 등록·수정·사용여부·삭제 기능
    combo 컴포넌트에 공통 코드 데이터를 바인딩하고, 각 콤보 그룹을 transaction으로 호출하여 데이터셋에 동적으로 로딩함으로써 화면 간 재사용성과 유지보수의 효율성을 확보
  • 상태변경 이력 관리 :
    사용자의 모든요청사항의 변경 전/후 값, 변경자 ID, 변경일시 등을 Oracle Database의 history 테이블에 저장
    시스템관리자만 이력 확인이 가능하며 필터 조건으로 기간, 사용자, 상태 등을 기준으로 검색 가능
