팀프로젝트 - 영화 리뷰를 활용한 감성 분석

  • 사용기술: python, keras, Html, crawling
  • 개발환경: Jupyter notebook
  • 프로젝트 기간: 2020.04~2020.06
  • URL: https://github.com/AgLine/sensitivity-analysis-

개발내용

영화 리뷰의 긍정, 부정, 중립을 구분하여 그래프로 알기쉽게 보여줍니다.

  • 전처리 :
    NULL, 공백, 영어제거
    불용어 제거
    okt.morphs를 사용하여 토큰화
  • LSTM :
    활성함수 softmax
    로스함수 categorical_crossentropy
    옵티마이저 rmsprop
  • 크롤링 :
    네이버 영화 리뷰 페이지 전체 크롤링
    Html parser을 통해 리뷰 부분만 가져옴
  • 워드클라우드 :
    리뷰의 키워드와 관련 없는 단어를 빼기 위한 불용어를 만듬
    많이 나오는 단어의 가중치를 줌
