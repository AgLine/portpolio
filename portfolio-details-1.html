팀프로젝트 - 수산물 위판정보 프로그램

  • 사용 기술:JAVA, SQL, 공공데이터포털 API, JDBC
  • 개발 환경: Eclipse, Oracle 11g
  • 프로젝트 기간: 2018.09~2018.11
  • URL: https://github.com/AgLine/marine-product-bill

개발내용

여러 사이트에서 정보를 찾을 필요 없이 실시간으로 단가를 알 수 있게 하여 좀 더 적절한 가격으로 효율적인 소비를 할 수 있게 도와줍니다.

  • 오픈api :
    Josn의 형태로 되어있는 데이터를 json parser를 이용하여 필요한 데이터만 파싱
  • 로그인 :
    소켓통신을 이용한 서버, 로그인과 로그아웃시 로그 저장
  • 회원가입 :
    회원정보 입력,필수입력정보를 입력하지 않거나 중복확인을 하지 않았을 경우 회원가입 불가능
    회원가입 시 데이터베이스에 정보 저장
  • 비밀번호 찾기 :
    비밀번호를 알려주지 않고 임시 비밀번호 100000~999999 랜덤으로 숫자 발급
  • 가격검색 :
    물고기를 검색하면 해당물고기의 정보를 데이터베이스에서 가져옴
    오픈api에서 파싱한 데이터를 가격정보에서 보여줌
