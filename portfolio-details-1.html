팀프로젝트 - 수산물 위판정보 프로그램

  • 사용 기술:JAVA, SQL, 공공데이터포털 API, JDBC
  • 개발 환경: Eclipse, Oracle 11g
  • 프로젝트 기간: 2018.09~2018.11
  • URL: https://github.com/AgLine/marine-product-bill

개발내용

효율적인 소비를 돕기 위해, 여러 사이트를 방문할 필요 없이 실시간으로 단가 정보를 확인할 수 있는 시스템을 개발했습니다. 이를 통해 사용자에게 더 적절한 가격으로 구매를 결정할 수 있는 도구를 제공합니다.

  • 오픈api :
    Josn의 형태로 되어있는 데이터를 json parser를 이용하여 필요한 데이터만 파싱
  • 로그인 :
    소켓 통신 기반 서버를 이용하여 로그인 및 로그아웃 시 활동 로그 저장
  • 회원가입 :
    사용자 정보 입력 시 필수 정보가 누락되거나, 중복 확인이 완료되지 않은 경우 회원가입 불가능
    성공적인 회원가입 시 사용자 정보를 데이터베이스에 저장
  • 비밀번호 찾기 :
    비밀번호를 제공하지 않고, 6자리 임시 비밀번호를 100000~999999 범위에서 랜덤으로 생성해 사용자에게 발급
    이후 비밀번호 재설정유도
  • 가격검색 :
    오픈 API에서 파싱한 단가 데이터를 사용자에게 직관적으로 보여줌
    검색된 물고기의 상세 정보를 데이터베이스에서 조회 및 제공
