팀프로젝트 - 온프레미스 네트워크 환경 구축

  • 사용기술: Rocky Linux 9, Apache, BIND, HAProxy, MySQL, PHP
  • 개발환경: VMware Workstation Pro, MobaXterm
  • 프로젝트 기간: 2025.04
  • URL: https://github.com/AgLine/On-Premises-Network-and-Server-Architecture

개발내용

내부망과 외부망을 분리하여 실제 온프레미스 환경을 시뮬레이션하고, DNS 서버를 통한 도메인 이름 관리, Apache 웹 서버 구축, HAProxy를 활용한 로드밸런싱 및 이중화 구성, PHP로 작성된 웹페이지에서 MySQL과의 연동을 통해 DB 조회 결과를 출력함. 외부 접속을 위한 포트포워딩 및 보안 설정을 함께 구성하여 전체 네트워크 아키텍처를 완성함

  • Router :
    ens160, ens224 네트워크 설정 및 방화벽 영역 분리
    DHCP 설정으로 내부 IP 자동 할당
    방화벽(http 서비스 허용) 및 포트포워딩(NAT) 설정
  • DNS 서버 :
    내부 도메인 wxforecast.gisang.chung
    정방향/역방향 영역(zone) 파일 작성
    내부 클라이언트의 DNS 요청 처리
  • Web 서버 :
    Apache 설치 및 index 페이지 구성
    내부망에서 DNS로 접근 가능 확인
  • HAProxy :
    roundrobin방식을 사용한 Web01,Web02서버 로드밸런싱
    option httpchk을 통해 웹서버 정상 작동 여부 확인
  • DB연동 :
    telnet명령어를 통해 3306포트가 열려있는지 확인 후 MySQL 접속
    웹서버에 작성한 db.php 파일을 통해 DB 쿼리 결과를 웹페이지로 출력
