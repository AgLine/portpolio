개인프로젝트 - JSP로 만든 게시판 사이트

  • 사용기술: JAVA, JSP, bootstrap, SQL
  • 개발환경: Eclipse, tomcat v8.5, mariaDB
  • 프로젝트 기간: 2023.03
  • URL: https://github.com/AgLine/bulletin-board-system-JSP

개발내용

jsp와 bootstrap을 사용하여 게시판을 만들었습니다.

  • Create :
    로그인된 사용자만 글쓰기 가능
  • Read :
    모든 사용자는 글을 확인할 수 있음
  • Update :
    session의 ID와 글 작성자가 동일할 경우 수정 버튼 활성화
  • Delete :
    session의 ID와 글 작성자가 동일할 경우 삭제 버튼 활성화
  • session :
    session을 통한 로그인, 로그아웃
